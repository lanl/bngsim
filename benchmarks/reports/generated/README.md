# Generated Benchmark Reports

This directory contains Markdown benchmark reports automatically generated by the
`bngsim/benchmarks/` suite emitters.

## Generation

Reports are created by running:

```bash
cd bngsim
python benchmarks/run_all.py --emit-only
```

Or to run benchmarks and generate reports:

```bash
python benchmarks/run_all.py
```

## LaTeX Mirroring

To mirror raw LaTeX fragments for a private paper repository, set the
`BNGSIM_BENCH_LATEX_DIR` environment variable:

```bash
export BNGSIM_BENCH_LATEX_DIR=/path/to/private/paper/latex/generated
python benchmarks/run_all.py --emit-only
```

This will write both:
- `reports/generated/<name>.md` (public Markdown reports)
- `$BNGSIM_BENCH_LATEX_DIR/<name>.tex` (private LaTeX fragments)

## Content

Each suite's `emit.py` reads `results/*.json` and writes a report here. The
reports use best-effort LaTeX-to-Markdown conversion for table and figure
formatting.
