# Kozer EGFR — problem 10 of the fitting benchmark

PyBioNetFit ODE fitting job on the Kozer et al. EGF receptor activation
model. This is the `kozer_egfr` row of the multi-model fitting benchmark
(see `../README.md`); it corresponds to problem 10 of the PyBNF
iScience-2019 example set.

## Model provenance and unit system

- **Kozer N et al. (2013)** *Mol BioSyst* 9: 1849-1863. The original ODE
  model, with concentrations in nM (intensive units).
- **Kozer N et al. (2014)** *Biochemistry* 53: 2594-2604. An
  NFsim-compatible reformulation with abundances in molecules per cell
  (extensive units).

`bngl/ode.bngl` uses the Kozer 2014 (molecules/cell) formulation as its
model body. The fitting job reproduces the Kozer 2013 fitting task —
fitting to the Kozer 2013 dose-response and time-course data — using
this 2014 model formulation.

The data files (`data/timecourse.exp`, `data/doseresponse.exp`) come
from the Kozer 2013 paper, processed for PyBioNetFit. Scaling factors
`alpha{1..4}_pre` translate the molecules/cell model outputs into the
data's measurement units; the parameter ranges in the `.conf` files are
taken from the canonical `lanl/PyBNF/examples/egfr_nf/egfr_nf.conf`.

## Files

- `bngl/ode.bngl` — network-based ODE. `generate_network` with
  `max_stoich={EGF=>4,EGFR=>4}` (truncates oligomers at tetramers, as in
  the Kozer 2013 ODE study), then `parameter_scan` and `simulate` with
  `method=>"ode"`. Every action runs to `t_end=600` s — the experimental
  readout time, 10 min after ligand addition.
- `conf/subprocess.conf` — `bngl_backend=bionetgen` (BNG2.pl / run_network).
- `conf/bngsim.conf` — `bngl_backend=bngsim` (in-process).
- `conf/_smoke/` — reduced-size confs for quick path re-verification.

Unlike the other problem dirs, these two confs are **hand-curated**
(`alpha` linear-scale parameters, Kozer-specific bounds) rather than
generated by `harness/make_problem_confs.py`. Both use `fit_type=ss`
with identical settings, so the only difference is the simulator path.

## Network-free Kozer jobs

The original single-model Kozer benchmark also had NFsim and RuleMonkey
jobs. Those were moved to `../_stash/kozer_networkfree/` when the
benchmark became ODE-only; see that directory's README.

## Simulation horizon

`t_end=600` s for every action — the experimental readout time. The
dose-response scan is deliberately *not* integrated to steady state,
because not all ligand conditions equilibrate within 10 minutes; fitting
at 600 s reproduces what the experiment measured.
